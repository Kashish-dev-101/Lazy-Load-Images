IntersectionObserver Native Lazy Loading

Native lazy loading demo

This demo showcases the browser-native loading="lazy" attribute for images. Instead of loading all images upfront, the browser defers off-screen images until the user scrolls near them—improving initial page load performance and reducing bandwidth.

How to observe: Open DevTools → Network tab, then scroll down slowly. You'll see images load progressively as they approach the viewport.
