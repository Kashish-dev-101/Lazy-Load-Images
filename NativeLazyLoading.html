Native lazy loading demo

This demo uses browser native loading="lazy".Browsers use predictive heuristics rather than strict visibility checks, loading images when they fall within a lookahead distance below the viewport. Large vertical spacers are added to ensure most images start outside this lookahead range, making the lazy loading behavior easy to observe.
